Java快速入门、IDEA开发工具的使用

Java概述

Java是sun公司1995年推出的，2009年被Oracle收购

Java的爸爸：詹姆斯.高斯林

Java是一门高级编程语言

流行度高，商业占用率高

重要特性：可移植性

Java用来干啥？

  • 万物皆可，最被市场认可的是企业级开发（如淘宝、京东这样的互联网系统）

Java的技术体系

  • JavaSE

    • 标准版：Java技术的核心和基础

  • JavaEE

    • 企业版：大型互联网企业级解决方案，充分被市场认可的

  • JavaME

    • 小型版：移动应用的解决方案，未被市场认可

Java产品

JDK

  • 开发者工具包，必须安装才可使用

Oracle官网

  • 下载，安装时路径避免使用中文和空格

常用的有JDK-8（企业常用）,JDK-11(学习使用),JDK-17

LTS：长期支持版本：JDK-8,JDK-11,JDK-17

JDK中重要的2个程序：

  • Javac

    • 编译程序

  • Java

    • 执行程序

JDK安装后验证是否安装成功

  • 打开命令行窗口

    • WIN+R—>cmd回车

  • 依次输入Java、Javac、Java-version…….判断

补充知识：常用命令行命令

  • cls

    • 清屏

  • cd

    • 进入或回退

      • cd xxx

        • 进入单级目录

      • cd x:\xxx\xxx

        • 进入多级目录

      • cd ..

        • 回退上一级目录

      • cd \

        • 回退到盘符根目录

  • dir

    • 查看当前路径下的文件信息

  • 盘符：

    • 切换到某个盘下（如D：）

开发一个Java入门程序：HelloWorld

1、编写代码

  • 建议全英文名称，首字母大写，后缀.java结尾
  • 代码：
    public class HelloWorld{
    public static void main(String[]args){
    System.out.println(“Hello World”);
    }
    }

2、编译代码

  • javac HelloWorld.java

    • 产生.class文件

      • HelloWorld.class

3、执行代码

  • java HelloWorld

    • 输出：Hello World

Java程序的执行原理

Java语言最后会翻译，解释成计算机能够识别的机器语言：000111…….以此驱动机器干活

BUG

  • 代表程序出现问题或者漏洞的意思

  • DEBUG

    • 解决问题的意思

Path环境变量，JAVA_HOME

作用

  • 记住程序的路径，方便在命令行的任意目录驱动程序

注意：新版本JDK安装时会自动配置Javac和Java的路径，可直接在命令行窗口使用；老版本则需手动配置Path环境变量

  • Path—>D:\Program Files\Java\jdk-11.0.13\bin

JAVA_HOME

  • D:\Program Files\Java\jdk-11.0.13

多学一招

  • Path—>D:\Program Files\Java\jdk-11.0.13\bin

  • Path—>%JAVA_HOME%\bin

    • 建议手动配置时使用此方法

开发工具：Intellij IDEA

集成开发工具（IDE工具）

  • 好处：工程结构化管理项目，有报错提醒，代码智能补全……

官网下载

  • 傻瓜式安装

IDEA开发第一个Java程序

  • 1、创建工程project（建议选择空工程），指派一个工程存放路径
  • 2、必须创建模块module
  • 3、创建一个包package
  • 4、创建一个类class

补充知识：IDEA常用快捷键

Java基础语法

注释

  • 作用：对程序进行解释说明，不影响程序执行

  • 分类

    • 单行注释: //——-

    • 多行注释： /* */

    • 文档注释： /** */

      • 注意：文档注释的内容可以提取到文档说明书中去

  • 为什么不影响程序执行？

    • 编译后的.class文件没有注释

字面量

  • 目的：告诉程序员数据在程序中该怎样书写

  • 分类

变量

  • 作用：内存中的一块区域，可以存储数据，且数据可变化
  • 格式：数据类型 变量名称 = 初始值；
  • = 赋值：从右往左执行

变量使用注意事项

  • 1.变量必须先声明才能使用
  • 2.什么类型的变量只能存放什么类型的数据
  • 3.变量从定义开始到‘}’有效，同一个有效范围内不能重新定义重名的变量
  • 4.变量定义的时候可以没有初始值，但使用的时候必须有初始值（如 int age;）

变量在计算机中的底层原理

  • 变量存储的是数据的二进制形式

  • 数据变二进制的规则

    • 逢二进一/除二取余

  • 文字怎么存储到计算机底层

    • ASCII编码表固定了常见字符对应数字

      • 存储字符对应二进制形式
      • ’a’–>97–>01100001
      • ‘A’–>65–>01000001
      • ‘0’–>48–>00110000

  • 图片、音视频，存储均为二进制形式

  • 补充知识

    • 二进制转换成十进制

    • 二进制转换成8进制

      • 01011101–>01 011 101–>135

    • 二进制转换成16进制

      • 01011101–>0101 1101–>5D

数据类型

  • 定义变量： 数据类型 变量名称 = 初始值；

  • 作用：约束变量只能存储什么类型的数据，帮助定义出不同类型的变量

  • 分类

    • 引用数据类型（后面详见）

      • String

    • 基本数据类型

      • 整型

        • byte：字节整型

          • 占1个字节内存

            • 范围：-128~127

        • short：短整型

          • 占2个字节内存

        • int：整型（默认）

          • 占4个字节内存

        • long：长整型

          • 占8个字节内存

            • 随便写整型字面量，默认是int类型，需在后面加上L/l

      • 浮点型（小数）

        • float：单精度

          • 占4个字节内存

            • 随便写小数字面量，默认是double类型，需在后面加上F/f

        • double：双精度

          • 占8个字节内存

      • 字符型

        • char

          • 占2个字节内存

      • 布尔型

        • boolean

          • 占1个字节内存

关键字、标识符

  • 关键字

    • Java自己使用的，不能用来取名字

  • 标识符

    • 规则：不能数字开头，不能使用关键字，区分大小写

      • 建议使用”驼峰式“命名法，如”studyJava“
